# 🚀 Projeto de Onboarding - Lista Geral de Tasks

Bem-vindo(a) ao seu projeto de onboarding! Para facilitar seu aprendizado e acompanhamento, aqui está a lista geral das tasks organizadas em duas etapas principais:

## 1️⃣ Backend (NestJS + MongoDB)

Você deve começar pelo backend, implementando todas as funcionalidades da API:

- Cadastro e login com JWT e Redis
- Guardas de autenticação
- CRUD completo do recurso principal (criação, listagem, edição e exclusão)

**O que esperamos ao final dessa etapa:**

- API 100% funcional, com autenticação e autorização completas
- Testes unitários das usecases e testes e2e com 100% de coverage
- Um collection do Postman configurado com todos os endpoints para fácil teste da API

## 2️⃣ Frontend (Next.js)

Após a API estar pronta e testada, comece o desenvolvimento do frontend:

- Tela de login com autenticação via API
- Interceptor para enviar o token JWT em todas as requisições
- Telas para listar, criar, editar e excluir itens
- Gerenciamento de estado de forms com react-hook-form e zod
- Feedbacks visuais e tratamento de erros

**O que esperamos ao final dessa etapa:**

- Frontend em Next.js integrado à API, com fluxo completo de login e CRUD
- Código limpo e organizado, com commits frequentes
- Apresentação agendada para demonstrar o funcionamento do app e API para o time

## ⚙️ Orientações Gerais

- Foque em commits pequenos e mensagens claras
- Sempre faça testes manuais e automatizados quando possível
- Tire dúvidas com seu padrinho(a) ou com o time a qualquer momento
- Priorize qualidade, legibilidade e boas práticas
- Para cada task abra um PR e mande para o seu padrinho para correção
- Divirta-se aprendendo e construindo algo real!
